# Linux Fundamentals Part 2

# SSH

Used to connect to another machine securely.

Example:
bash
ssh user@ip


---

# ls -lh

Shows human readable file sizes.

Example:
bash
ls -lh


---

# User Switching

## su

Switch user.

Example:
bash
su user2
su - tryhackme


---

# Permissions

Linux permissions decide:
- who can read
- who can write
- who can execute

---

# Important Learnings
- Directories and files are different
- cat works only on files
- ls / shows root folders
- cd /etc moves into etc folder
- / is root directory

---

# Root Folders

## /home
User folders

## /etc
Configuration files

## /var
Logs and variable data

## /usr
Programs and tools
