# What is Networking?

Networking means connecting devices together so they can communicate and share data.

Examples of devices:
- Computers
- Phones
- Servers

Example flow:
Browser (Client) -> Sends Request  
Server -> Sends Response

---

# Types of Networks

## LAN (Local Area Network)
- Small area network
- Used in:
  - Home
  - College
  - Office

## WAN (Wide Area Network)
- Large area network
- Example:
  - Internet

---

# Ping

Ping checks:
- Device reachable aa?
- Internet working aa?
- Delay/latency entha?

Protocol used:
ICMP (Internet Control Message Protocol)

Command:
ping <ip>

Example:
ping 8.8.8.8

Working:
1. Echo Request send chestadhi
2. Echo Reply receive avthadhi

Used for:
- Troubleshooting
- Connectivity testing

---

# IP Address

IP address identifies a device in network.

Example:
192.168.1.1

IPv4:
- 4 octets
- Each octet range:
0-255

IP address can change.

---

# Types of IP Address

## Private IP
Used inside local network.

Examples:
- 192.168.x.x
- 10.x.x.x

Not visible on internet.

---

## Public IP
Given by ISP.

Used for internet identity.

Multiple devices can use same public IP through router/NAT.

---

# IPv4 vs IPv6

## IPv4
- Limited addresses
- Around 4.3 billion

Example:
192.168.1.1

## IPv6
- Huge address space
- Uses hexadecimal format

Future internet standard.

---

# MAC Address

Permanent physical address of device.

Example:
a4:c3:f0:85:ac:2d

Structure:
- First half -> Vendor/company
- Second half -> Unique device ID

Normally MAC address won't change.

---

# MAC Spoofing

Using fake MAC address.

Uses:
- Hide identity
- Bypass restrictions
