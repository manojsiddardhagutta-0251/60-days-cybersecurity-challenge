## Web Infrastructure

### Web Servers
Software handling website requests.

Examples:
- Nginx
- Apache

### Root Directory
Location storing website files.

Linux examples:
- /var/www/html

### Static Content
Same for all users.

Examples:
- Images
- CSS
- Static HTML

### Dynamic Content
Generated in real-time.

Examples:
- User dashboards
- Search results
- Social media feeds

### Virtual Hosts
Single server hosting multiple websites.

### CDN
Content Delivery Network

Stores cached content globally for faster delivery.

### Load Balancer
Distributes traffic across multiple servers.

Benefits:
- High availability
- Better performance
- Scalability

### Databases
Store application data.

Examples:
- MySQL
- PostgreSQL
- MongoDB

### WAF
Web Application Firewall

Filters malicious web traffic.

### Extra Information

### Why Infrastructure Matters
Understanding infrastructure helps during:
- Web exploitation
- Reconnaissance
- Bug bounty
- Pentesting

### Real World Attacks
Attackers target:
- Misconfigured servers
- Exposed databases
- Weak WAF rules
- Leaked admin panels
