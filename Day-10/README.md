## Extra Information Learned

### HTTPS Does Not Hide Everything
Attackers may still see:
- IP addresses
- Packet size
- Timing patterns
- Destination domains
- *Only actual content remains encrypted.*

### Importance of Cookies
Cookies are used for:
- Login sessions
- Authentication persistence
- User tracking
- Shopping carts

### Why HTTP Headers Matter
Headers may leak:
- Browser version
- Operating system
- Server software
- Authentication tokens

### Difference Between Static and Dynamic Content
- **Static Content:** Same for all users, faster, and easier caching.
- **Dynamic Content:** Generated in real time, personalized, and database-driven.
